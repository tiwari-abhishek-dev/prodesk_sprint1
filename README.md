# Prodesk IT Landing Page

A responsive landing page built using HTML, CSS, and JavaScript as part of Sprint 1 at Prodesk IT.

##Live Website

[View Website](https://prodesk-sprint1.vercel.app/)

[View Explaination_video drive link](https://drive.google.com/file/d/1qYZcVLcblYxKF1AZnNEe62KcU4TdwtRv/view?usp=drive_link)
##  Features
* Responsive sticky navigation bar
* Hero Section
* Call-To-Action Button
* Services section with 3 Cards
* Mobile responsive design
* Hover effects
* Dark mode toggle

## Technologies Used

* HTML
* CSS
* JavaScript
* Flexbox
* Media queries

## Responsive Design
* Desktop
* Tablet
* Mobile Devices
